check your content with OOPSPam API

OOPSpam detects the likelihood of spam more accurately. It allows you to scan your content utilizing multiple machine learning based, powerful text analyzers.

Oh, and for free!

Request:

curl -X POST \
      https://oopspam.herokuapp.com/ \
      -H 'cache-control: no-cache' \
      -H 'content-type: application/json' \
      -d '"text":"The example content!"}'

Response:

          {
            "spam": 2,
            "nospam": 1
          }

No data storing happens on our side. Currently, the API supports 3 spam filters including Apache SpamAssassin, will add more. Let me know if something comes up ❤
