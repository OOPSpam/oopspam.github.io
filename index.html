check your content with OOPSPam API

OOPSpam detects the likelihood of spam more accurately. It allows you to scan your content utilizing multiple machine learning based, powerful text analyzers.

Oh, and for free!

how to use

Send a POST request to

https://oopspam.herokuapp.com
with a JSON object containing your content
{"text":"The example content!"}
you will get something like this
{ "spam": 2, "nospam": 1 }

Example


curl -X POST \
      https://oopspam.herokuapp.com/ \
      -H 'cache-control: no-cache' \
      -H 'content-type: application/json' \
      -d '{"text":"The example content!"}'

Response:

          {
            "spam": 2,
            "nospam": 1
          }

Currently, the API supports 3 spam filters including Apache SpamAssassin. Let me know if something comes up.
